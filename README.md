# ArgyHub ReWork - Telegram Userbot
**ArgyHub** - небольшой Telegram юзербот с возможностью трекинга, аналитики или просто развлекухи.

> *Юзербот все еще в разработке и далек от идеала!*

Этот юзербот очень просто написан, в нем нет какого либо особенного кода и сделан чисто для опыта. Для вас он может быть как развлекательным, так и удобным в плане аналитики или просто облегчающим, делая общение или что нибудь другое в Telegram чуточку проще.

**Я не хочу нести ответственность за ваши действия, которые вы сделали в чатах или каналах, используя ArgyHub. Просьба: не злоупотребляйте некоторыми командами!**

**ArgyHub** на данный момент полностью работает в консольном режиме и взаимодействует с `Telegram API`.

## Установка

1. Скачайте и распакуйте `ZIP`-архив, который вы скачали с [Releases](https://github.com/MrArgy/ArgyHubRework/releases) (рекомендуется последняя версия)
2. Запустите `.EXE` файл и следуйте инструкциям, которые указаны в консоли.


Не доверяете `.EXE`? Тогда следующие пункты как раз для вас:

---

### Установка зависимостей

1. Скачайте `Python` последней версии, скачать можно [здесь.](https://www.python.org/downloads/)
2. Скачайте `Git` последней версии, скачать можно [здесь.](https://git-scm.com/)
3. Создайте папку, куда вы будете распаковывать **ArgyHub**.
4. Клонируйте туда репозиторий следующей командой:
```bash
git clone https://github.com/MrArgy/ArgyHubRework.git
cd ArgyHubRework
```
5. Установите все нужные плагины следующей командой (без них скрипт не запустится):
```bash
pip install -r requirements.txt
```
6. Откройте `main.py` и найдите строчки `API_ID`, `API_HASH` и вставьте свое.
7. Потом вводите команду запуска в командой строке:
```bash
python main.py
```
или
```bash
py main.py
```
8. Готово! Используйте на здоровье.
> Есть вопросы? Пишите [сюда!](https://t.me/argyy)

### Получение API_ID и API_HASH

1. Заходите на [my.telegram.org](https://my.telegram.org/) и вводите там свой номер и вставляете код в пустое поле под номером.
2. Потом нажимаете на [API development tools](https://my.telegram.org/apps).
3. Копируйте `API_ID` на `API_HASH` и вставляйте в строчки, где указан `API_ID` и `API_HASH`.
> Есть вопросы? Пишите [сюда!](https://t.me/argyy)

### Билд
1. Если вы хотите самостоятельно забилдить **ArgyHub**, вы можете использовать `PyInstaller`, вам нужно это установить следующей командой:
```bash
pip install pyinstaller
```
2. После этого, запустите скрипт `BUILD.py` *(у вас должны быть установлены все библиотеки из `requirements.txt`!)*, следующей командой:
```bash
python BUILD.py
```
или
```bash
py BUILD.py
```
3. После этого, у вас начнется билд, если вы билдите впервые, это может занять от 3 до 10 минут.
4. Как только у вас забилдиться, у вас создатся папка `dist`, в ней лежит готовый `.EXE`, запускайте его и готово!
