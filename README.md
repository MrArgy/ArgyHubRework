# ArgyHub ReWork - Telegram Userbot
ArgyHub - небольшой Telegram юзербот с возможностью трекинга, аналитики или просто развлекухи.

> Юзербот все еще в разработке и далек от идеала!

ArgyHub на данный момент полностью работает в консольном режиме и взаимодействует с Telegram API.

## Установка

1. Скачайте и распакуйте ZIP-архив, который вы скачали с [Releases](https://github.com/MrArgy/ArgyHubRework/releases) (рекомендуется последняя версия)
2. Запустите .EXE файл и следуйте инструкциям, которые указаны в консоли.
